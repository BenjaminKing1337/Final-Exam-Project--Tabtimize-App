# Tabtimize App

## 1. Install project
```
npm install
```

### 2. Hot reload for development
```
npm run serve
```

### 3. Minifies for production
```
npm run build
```

### 4. Run your tests if necessary
```
npm run test
```

### 5. Linting and fixing
```
npm run lint
```
